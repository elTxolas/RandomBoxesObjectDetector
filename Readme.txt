This archive contains ring_front_center camera frames from Argoverse-1.1 and the corresponding COCO-style annotations from Argoverse-HD.

Note that "test-meta.json" only includes a list of frames but not the annotations, which is held out for the streaming perception challenge:
https://eval.ai/web/challenges/challenge-page/800/overview

For more infomation about Argoverse-1.1, please visit https://www.argoverse.org/data.html.
For more infomation about Argoverse-HD, please visit https://www.cs.cmu.edu/~mengtial/proj/streaming/.
